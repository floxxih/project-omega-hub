# StackHub Smart Contracts

This directory contains the Clarity smart contracts for the StackHub platform.

## Contracts

- `stackhub-nft-marketplace.clar`: NFT trading logic.
- `stackhub-token-launchpad.clar`: Token creation factory.
- `stackhub-staking-vault.clar`: Staking and rewards.
- `stackhub-service-registry.clar`: On-chain service directory.

## Development

Run tests:
```bash
npm test
```
